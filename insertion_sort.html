插入排序法 Insertion Sort

理解原理、偽代碼、複雜度，並透過視覺化逐步演練

1. 演算法原理

插入排序會把陣列分成「已排序區間」與「未排序區間」。每一輪取出未排序區間的第一個元素（key），向左比較並插入到正確位置。

特性：穩定排序、原地操作，對接近排序完成的資料特別有效率。

2. 偽代碼 

INSERTION-SORT(A)
  for i = 1 to n-1
      key = A[i]
      j = i - 1
      while j >= 0 and A[j] > key
          A[j+1] = A[j]
          j = j - 1
      A[j+1] = key
  return A

C 語言參考實作 

#include <stdio.h>

void insertion_sort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;

        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }
}

int main(void) {
    int arr[] = {7, 3, 9, 2, 5};
    int n = sizeof(arr) / sizeof(arr[0]);

    insertion_sort(arr, n);

    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}

3. 複雜度

項目結果說明
平均時間O(n^2)一般情況下每輪都可能向左比對多個元素並搬移，整體為二次方等級。
最壞時間O(n^2)當資料接近反序時，每次插入都要移動已排序區間的大部分元素。
最佳時間O(n)資料幾乎已排序時，每輪只需一次比較即可完成插入。
空間複雜度O(1)僅使用少量暫存變數，不需額外陣列。
穩定性穩定（Stable）對相等元素不會跨越交換，可保留原本相對順序。
