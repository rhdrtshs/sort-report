1. 演算法原理

桶排序會先根據數值範圍把資料分散到多個桶（bucket）中，再對每個桶內元素排序，最後依桶的順序串接回原陣列。

特性：若資料分布均勻，平均時間可接近線性；效能受資料分布與桶內排序方法影響。

2. 偽代碼

BUCKET-SORT(A)
  create k empty buckets
  for each x in A
      put x into bucket[index(x)]
  for i = 0 to k-1
      sort bucket[i]
  concatenate bucket[0], bucket[1], ... , bucket[k-1]

3. 複雜度

平均時間最壞時間最佳時間空間
$O(n + k)$ (分布均勻)$O(n^2)$ (集中於同桶)$O(n + k)$$O(n + k)$
