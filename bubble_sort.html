1. 演算法原理

氣泡排序每一輪從左到右比較相鄰元素，若順序錯誤就交換。每完成一輪，當前最大值會被「推」到未排序區間最右端。

特性：穩定排序、原地操作，最佳情況可藉由提早終止達到較佳效能。

2. 偽代碼

BUBBLE-SORT(A)
  for i = 0 to n-2
      swapped = false
      for j = 0 to n-i-2
          if A[j] > A[j+1]
              swap A[j], A[j+1]
              swapped = true
      if not swapped
          break
  return A
