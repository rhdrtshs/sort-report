1. 演算法原理

氣泡排序每一輪從左到右比較相鄰元素，若順序錯誤就交換。每完成一輪，當前最大值會被「推」到未排序區間最右端。

特性：穩定排序、原地操作，最佳情況可藉由提早終止達到較佳效能。

2. 偽代碼 

BUBBLE-SORT(A)
  for i = 0 to n-2
      swapped = false
      for j = 0 to n-i-2
          if A[j] > A[j+1]
              swap A[j], A[j+1]
              swapped = true
      if not swapped
          break
  return A

C 語言參考實作

#include 

void bubble_sort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int swapped = 0;

        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = 1;
            }
        }

        if (!swapped) {
            break;  // Early break: 本輪無交換，表示已排序完成
        }
    }
}

int main(void) {
    int arr[] = {7, 3, 9, 2, 5};
    int n = sizeof(arr) / sizeof(arr[0]);

    bubble_sort(arr, n);

    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}

3. 複雜度

項目結果說明
平均時間O(n^2)隨機資料下通常需要多輪比較與交換，整體比較次數約為二次方等級。
最壞時間O(n^2)當資料接近反序時，每一輪都會大量交換，達到最大比較與交換成本。
最佳時間O(n)若序列已排序，第一輪檢查到無交換即可提早終止，只需線性掃描。
空間複雜度O(1)只使用固定數量的暫存變數（如 swap 暫存），不需額外陣列。
穩定性穩定（Stable）相等元素僅在「大於」時才交換，因此相對順序可被保留。
