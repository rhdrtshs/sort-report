氣泡排序法 (Bubble Sort) 學習筆記

1. 演算法簡介

氣泡排序法是一種簡單的排序演算法。它重複地走訪過要排序的數列，一次比較兩個元素，如果它們的順序錯誤（如大到小）就把它們交換過來。走訪數列的工作是重複地進行直到沒有再需要交換，也就是說該數列已經排序完成。

這個演算法的名字由來是因為越小的元素會經由交換慢慢「浮」到數列的頂端。

2. 過程說明與範例

假設我們有一個陣列：[5, 1, 4, 2]

步驟 比較對象 狀態 說明
第一輪 5 vs 1 [1, 5, 4, 2] 5 > 1，進行交換
第一輪 5 vs 4 [1, 4, 5, 2] 5 > 4，進行交換
第一輪 5 vs 2 [1, 4, 2, 5] 5 > 2，進行交換。5 已固定在末尾。

3. 程式碼呈現 (JavaScript)


function bubbleSort(arr) {
    let n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        for (let j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交換元素
                let temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    return arr;
}

4. 複雜度呈現

分析項目 複雜度 (Big O)
平均時間複雜度 $O(n^2)$
最壞時間複雜度 $O(n^2)$
最佳時間複雜度 $O(n)$ (已排序時)
空間複雜度 $O(1)$ (原地排序)
