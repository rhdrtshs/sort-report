氣泡排序（Bubble Sort）

簡介

氣泡排序會重複走訪陣列，比較相鄰元素並在需要時交換，直到陣列排好。它簡單但效率低，通常只用於教學或小資料量。

複雜度
  • 時間複雜度：最差/平均 O(n^2)，最佳 O(n)（若加入已排序檢查）
  • 空間複雜度：O(1)（原地排序）
  • 穩定性：穩定（相等元素順序不改變）
演算法步驟（概要）
  1. 從陣列左端開始，依次比較相鄰兩個元素。
  2. 若左邊元素大於右邊元素，交換它們。
  3. 重複一次完整走訪（稱為一輪），每輪會把一個最大的元素放到右端。
  4. 重複多輪，直到一整輪沒有交換為止（表示已排序）。

範例陣列與逐步說明

範例：[5, 3, 8, 4, 2]

  1. 第一輪比較並交換：[3,5,4,2,8]（8 移到最後）
  2. 第二輪：[3,4,2,5,8]
  3. 第三輪：[3,2,4,5,8]
  4. 第四輪：[2,3,4,5,8]（完成）
視覺化與互動

比較時為 橙色，交換時為 紅色。
