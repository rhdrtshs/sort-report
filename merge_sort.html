合併排序法 Merge Sort

理解分治法核心概念，並透過視覺化追蹤拆分與合併過程

回首頁

1. 演算法原理

合併排序使用分治法：先把陣列遞迴切成更小的子陣列，直到每段只剩一個元素，再將已排序的小段兩兩合併成較大的有序段。

特性：穩定排序、時間複雜度穩定為 $O(n \log n)$，但需要額外陣列空間。

2. 偽代碼 

MERGE-SORT(A, l, r)
  if l >= r
      return
  m = floor((l + r) / 2)
  MERGE-SORT(A, l, m)
  MERGE-SORT(A, m+1, r)
  MERGE(A, l, m, r)

MERGE(A, l, m, r)
  L = A[l..m], R = A[m+1..r]
  merge L and R back into A[l..r] in sorted order

C 語言參考實作 

#include <stdio.h>

void merge(int arr[], int left, int mid, int right) {
    int n1 = mid - left + 1;
    int n2 = right - mid;
    int L[n1], R[n2];

    for (int i = 0; i < n1; i++) L[i] = arr[left + i];
    for (int j = 0; j < n2; j++) R[j] = arr[mid + 1 + j];

    int i = 0, j = 0, k = left;
    while (i < n1 && j < n2) {
        if (L[i] <= R[j]) arr[k++] = L[i++];
        else arr[k++] = R[j++];
    }
    while (i < n1) arr[k++] = L[i++];
    while (j < n2) arr[k++] = R[j++];
}

void merge_sort(int arr[], int left, int right) {
    if (left >= right) return;
    int mid = left + (right - left) / 2;
    merge_sort(arr, left, mid);
    merge_sort(arr, mid + 1, right);
    merge(arr, left, mid, right);
}

int main(void) {
    int arr[] = {7, 3, 9, 2, 5};
    int n = sizeof(arr) / sizeof(arr[0]);

    merge_sort(arr, 0, n - 1);

    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
    return 0;
}

3. 複雜度

項目結果說明
平均時間O(n log n)每層合併需線性處理全部元素，遞迴深度約為 log n。
最壞時間O(n log n)無論資料初始順序如何，切分與合併流程都固定進行。
最佳時間O(n log n)即使資料已排序，仍需完成遞迴切分與合併步驟。
空間複雜度O(n)合併過程需額外暫存左右子陣列來寫回結果。
穩定性穩定（Stable）合併時先取左段相等元素，可保留相對順序。
