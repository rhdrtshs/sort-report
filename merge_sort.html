1. 演算法原理

合併排序使用分治法：先把陣列遞迴切成更小的子陣列，直到每段只剩一個元素，再將已排序的小段兩兩合併成較大的有序段。

特性：穩定排序、時間複雜度穩定為 $O(n \log n)$，但需要額外陣列空間。

2. 偽代碼

MERGE-SORT(A, l, r)
  if l >= r
      return
  m = floor((l + r) / 2)
  MERGE-SORT(A, l, m)
  MERGE-SORT(A, m+1, r)
  MERGE(A, l, m, r)

MERGE(A, l, m, r)
  L = A[l..m], R = A[m+1..r]
  merge L and R back into A[l..r] in sorted order
