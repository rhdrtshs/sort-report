選擇排序法 Selection Sort

理解原理、偽代碼、複雜度，並透過視覺化逐步演練

1. 演算法原理

選擇排序每一輪都在未排序區間中「找最小值」，再與該區間的第一個元素交換。完成一輪後，最小值會固定在正確位置，未排序區間向右縮小。

特性：不穩定排序、原地操作、不依賴額外資料結構。

2. 偽代碼

SELECTION-SORT(A)
  for i = 0 to n-2
      min = i
      for j = i+1 to n-1
          if A[j] < A[min]
              min = j
      if min != i
          swap A[i], A[min]
  return A
